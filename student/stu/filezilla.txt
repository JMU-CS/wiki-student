====== Transferring Files from Your JMU CS Account with FileZilla ======

  - Install the [[https://filezilla-project.org/download.php?type=client|FileZilla client]]
  - Open FileZilla
  - If the Site Manager doesn't automatically open, click the icon in the top left of the FileZilla window (see #1 in image) to open it
    * Main FileZilla Window\\ {{:student:stu:filezilla_-_main_window.png?direct&800|}}
        - Site Manager button
        - the files on the computer you're currently using (i.e. your laptop, called the "local" computer)
        - the files on "stu" the JMU CS server that hosts your account (the computer that you're not currently using, but have connected to is the "remote" computer)
  - In the Site Manager window, click the New site button (annotated with the ellipse in the figure)
      * FileZilla Site Manager without info\\ {{:student:stu:filezilla_-_site_manager_-_init.png?direct&800|}}

      - Logon Type: Ask for Password
