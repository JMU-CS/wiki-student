# Cluster

The JMU CS/CISE computational cluster is located in the Frye data center and is currently comprised of 17 Dell PowerEdge servers with dual 16-core 3.1GHz AMD EPIC processors and 64 GB RAM each. Eight of the servers also have a single NVIDIA A2 GPU. The cluster is used for teaching CS 470 and doing research. For more information, contact Mike Lam.
